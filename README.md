## 🌟 AI & Algorithm Projects

### 🥘 내 냉장고를 부탁해 (My Fridge): Intelligent Recipe System
> **식재료 인식부터 자연어 기반 레시피 추천까지 제공하는 지능형 서비스**

* **Computer Vision & Mobile Deployment**: 
    * 냉장고 속 식재료 자동 인식을 위한 **Object Detection 모델** 학습 및 추론 파이프라인 구축.
    * 모바일 환경 최적화를 위해 학습된 모델을 **TFLite 형식으로 Export**하여 안드로이드 앱 탑재.
* **LLM 기반 데이터 엔지니어링**:
    * 유튜브 등 외부 플랫폼의 비정형 레시피 데이터에서 **LLM(Large Language Model)을 활용**하여 핵심 정보를 정형화된 형태로 자동 추출하는 시스템 구축.
* **지능형 검색 및 추천 알고리즘**:
    * **Elasticsearch의 'nori' 한글 분석기**와 가중치 스코어링을 활용하여 "칼칼한", "매콤한" 등 감성 키워드 기반의 전문(Full-text) 검색 시스템 고도화.
    * 사용자 보유 재료와의 매칭 비율(Ratio)을 계산하여 최적의 레시피를 제안하는 추천 로직 구현.
* **Stack**: `FastAPI`, `Elasticsearch`, `TFLite`, `Python (Crawler)`, `PostgreSQL`.

### 🎓 T-LECTURE: Optimization-based Management System
> **다중 제약 조건을 고려한 알고리즘 기반 강사 자동 배정 솔루션**

* **Optimization Algorithm Design**:
    * 거리 제한, 중복 방지 등의 **Hard Filter**와 형평성, 거리 점수 등 9가지 변수를 조합한 **Soft Scoring 가중치 모델** 기반의 강사 배정 최적화 알고리즘 직접 설계.
* **Automated Data Pipeline**:
    * Kakao Maps API를 활용해 주소를 좌표로 변환하고, 강사-교육 장소 간의 **실제 도로 주행 거리**를 자동으로 계산 및 갱신하는 배치 시스템 구축.
* **Infrastructure Monitoring**:
    * GitHub Actions를 활용해 서버 및 DB의 가용성을 상시 모니터링하고 헬스체크를 수행하는 자동화된 운영 환경 구축.
* **Stack**: `Node.js`, `Prisma`, `PostgreSQL`, `GitHub Actions`, `Kakao Maps API`.
